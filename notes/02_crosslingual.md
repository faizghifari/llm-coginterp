# Cross-Lingual Evaluation

This document categorizes benchmarks related to cross-lingual evaluation.

## Cross-Lingual Benchmarks

### Cross-lingual Choice of Plausible Alternatives (`xcopa`) [Link](https://arxiv.org/abs/2005.00333)
**Subcategory:** commonsense-reasoning

Covers low-resource languages (Quechua, Haitian Creole); tests causal reasoning

### Cross-lingual Natural Language Inference (`xnli`) [Link](https://arxiv.org/abs/1809.05053)
**Subcategory:** NLI

Core benchmark for cross-lingual NLI; widely used since 2018

### Cross-lingual Question Answering Dataset (`xquad`) [Link](https://arxiv.org/abs/1910.11856)
**Subcategory:** extractive-QA

240 paragraphs + 1190 Q-A pairs from SQuAD v1.1; professional translations

### Cross-lingual Transfer Evaluation of Multilingual Encoders (`xtreme`) [Link](https://arxiv.org/abs/2003.11080)
**Subcategory:** multi-task

Includes XNLI, XQuAD, MLQA, PAN-X, PAWS-X sub-tasks
